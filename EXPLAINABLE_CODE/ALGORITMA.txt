================================================================================
                         KERNEL SHAP - SIMPLE VERSION
================================================================================

APA ITU KERNEL SHAP?
Algoritma untuk jelaskan kenapa model ngasih prediksi tertentu.
Caranya: hitung kontribusi setiap kata/token → berapa banyak ngaruh ke hasil.

HASIL:
  Nilai POSITIF  = kata naikin prediksi (good untuk label ini)
  Nilai NEGATIF  = kata turunin prediksi (bad untuk label ini)
  Nilai KECIL    = kata gk ngaruh banyak


================================================================================
                              FUNGSI - FUNGSI
================================================================================

1. PREDICT_PROBA(x_vec, clf, target_index)
   ───────────────────────────────────
   Fungsi: Minta model berapa probabilitas untuk kelas target?
   Input: vektor kata, model, index kelas yang mau dijelasin
   Output: nilai probabilitas (0.0 - 1.0)


2. PREDICT_ON_SUBSET(subset, x_instance, clf, target_index)
   ──────────────────────────────────────────────────
   Fungsi: Prediksi kalau HANYA kata-kata tertentu saja yang aktif
   
   Contoh konkret:
   - Asli: "saya suka kamu" (3 kata)
   - Subset [saya, kamu]: model prediksi untuk "saya ___ kamu"
   - Subset []: model prediksi untuk "_____ _____ _____" (kosong)
   
   Jadi kita lihat: kalau hilang kata X, prediksi berubah berapa?


3. KERNEL_WEIGHT(M, k)
   ──────────────────
   Fungsi: Berapa penting subset ini untuk perhitungan?
   
   Subset kosong sama penuh dianggap gk penting (bobot kecil)
   Subset dengan beberapa kata dianggap penting (bobot besar)


================================================================================
                        ALGORITMA KERNEL SHAP (5 STEP)
================================================================================

INPUT:
  - Kalimat yang mau dijelasin
  - Vectorizer (TF-IDF atau Count)
  - Model SVM/LR
  - List nama-nama fitur
  - List nama-nama label
  - Mana label yang mau dijelasin?


STEP 1: PERSIAPAN
─────────────────
1. Vectorisasi kalimat → dapat vektor sparse
2. Ambil kata-kata yang aktif (ada dalam kalimat)
3. Catat berapa jumlah kata aktif = M

Contoh:
  Kalimat: "saya benci kamu"
  Kata aktif: ['saya', 'benci', 'kamu']
  M = 3


STEP 2: BUAT SEMUA KOMBINASI KATA
──────────────────────────────────
Generate berbagai subset dari kata-kata aktif.

Kalau M kata, maka ada 2^M kemungkinan subset:
- Subset 0 kata: []
- Subset 1 kata: [saya], [benci], [kamu]
- Subset 2 kata: [saya,benci], [saya,kamu], [benci,kamu]
- Subset 3 kata: [saya,benci,kamu]

Total = 8 subset

Jika terlalu banyak → random sample yang terpenting (wajib ambil kosong dan penuh)


STEP 3: PREDIKSI SETIAP SUBSET
──────────────────────────────
Untuk setiap subset:
  1. Turunin prediksi model (pakai predict_on_subset)
  2. Hitung bobot subset (pakai kernel_weight)
  3. Buat mask: 1 jika kata ada, 0 jika gk ada

Hasilnya:
  - X_mask: setiap baris = 1 subset (format: [bias, 1/0, 1/0, 1/0])
  - y: prediksi untuk setiap subset
  - W: bobot untuk setiap subset


STEP 4: CACAH KONTRIBUSI SETIAP KATA (LINEAR REGRESSION)
──────────────────────────────────────────────────────────
Pakai formula weighted least square:
  β = (X.T @ W @ X)^(-1) @ X.T @ W @ y

Output:
  - β[0] = baseline (ekspektasi tanpa tahu apapun)
  - β[1:] = kontribusi setiap kata (SHAP value)


STEP 5: HASIL & CETAK
─────────────────────
1. Urutkan kata berdasarkan |kontribusi| (terbesar duluan)
2. Tampilkan:
   - Kalimat asli
   - Setiap kata + kontribusinya
   - Baseline
   - Validasi: baseline + Σ(kontribusi) ≈ prediksi asli


================================================================================
                          CARA BACA HASIL
================================================================================

Baseline (E[f(x)]):
  Probabilitas rata-rata model untuk kelas ini (gk tahu apa-apa)
  
SHAP Value untuk setiap kata:

  POSITIF  → Kata dukung prediksi (push naik) 
             Semakin besar = semakin strong dukungnya
  
  NEGATIF  → Kata lawan prediksi (push turun)
             Semakin besar (negatif) = semakin strong lawannya
  
  NKECIL   → Kata gak penting, gak ngaruh banyak


RUMUS AKHIR:
  Prediksi = Baseline + Σ(SHAP values untuk setiap kata)

Kalau ada kata dengan SHAP +0.15, artinya kata itu naikin prediksi 0.15 poin.


================================================================================
                          PLOTTING - BACA GRAFIK
================================================================================

Hasil plot:
  - Bar horizontal untuk setiap kata (top 10)
  - Merah = kontribusi positif (naikin prediksi)
  - Biru = kontribusi negatif (turunin prediksi)
  - Garis putus = baseline
  - Text box = prediksi final + baseline

Cara baca:
  Bar merah paling panjang = kata paling membantu prediksi kelas ini
  Bar biru paling panjang = kata paling menghalangi prediksi kelas ini
